# 车型图放这里

把车型图片放到本目录，文件名对应剧本里的 image 字段，网页对比卡/工具卡会自动显示：

| 文件名 | 用在哪个客户/车型 |
|---|---|
| tank300.jpg   | 坦克300（张先生/李女士/周先生） |
| havalh6.jpg   | 哈弗H6（陈女士/王先生/吴女士） |
| havaldog.jpg  | 哈弗大狗（赵先生/孙女士） |

- 建议尺寸：宽 ≥ 320px，横图（卡片按 108×78 裁切显示）。
- 缺图不影响演示：网页会显示一个占位框，提示该放哪张图。
- 想加别的车型：在 demo_scenarios.py 里给对应剧本的 image 填新文件名，再把图放进来即可。
